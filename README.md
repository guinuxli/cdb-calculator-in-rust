## 🇺🇸 English Version

This is **my first real Rust 🦀 project** — a simple but functional **CDB calculator**!

Since I've been learning Rust for **less than a month**, I wanted to start with something simple — but **not generic**. A common calculator felt too plain, so I decided to build a **CDB calculator** instead. It forces you to think a little harder about **how the Rust language actually works**, which made it a much more interesting challenge. 🧠

The CLI calculator is **100% functional** and works for real — you could actually use it without any issues. Of course, it doesn't replace a robust financial calculator, but it **shows off how Rust works** in a practical and meaningful way. 🦀⚙️

---

### ✨ Features

- 🦀 Built entirely in **Rust**
- 💰 Calculates **CDB returns** based on CDI rate
- 🖥️ Fully functional **CLI interface**
- 🧪 Great for understanding Rust fundamentals in a real-world context

---

### 🚀 Getting Started

```bash
git clone https://github.com/guinuxli/cdb-calculator-in-rust.git
cd cdb-calculator-in-rust
cargo run
```

This project was born out of a desire to **learn by doing** — picking a problem that's simple enough to be approachable, but interesting enough to keep things fun. If you're also just starting out with Rust 🦀, I hope this inspires you to build something **real** too!
